### Prerequisites
This project use YARN to manage Node package
```
npm install --save yarn
```
### Installing
At first, you have to install all package
```
yarn install
```
And then, the project is ready.<br>
<li>To run dev server</li>

```
yarn start
```
<li>To build</li>

```
yarn build
```
